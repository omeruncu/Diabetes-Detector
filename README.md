## About The Project


## Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/omeruncu/Diabetes-Detector.git
   ```
2. Switch to the project folder:
   ```bash
   cd Diabetes-Detector
   ```
3. Install the required Python libraries:
   ```bash
   pip install -r requirements.txt
   ```
